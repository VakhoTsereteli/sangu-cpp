# 8.ნულოვანი ეფექტი

ნულოვანი ეფექტი არის როდესაც პროგრამაში გვიწერია გამოსახულება რომელიც რაიმე
შედეგს აბრუნებს და ჩვენ ამ შედეგს არ ვიყენებთ და არც ვინახავთ.
მაგალითად:

```cpp
#include <iostream>

int square(const int& x) {
    return x * x;
}

int main() {
    square(5); // შედეგი იკარგება
}
```
